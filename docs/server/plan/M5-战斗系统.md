## 5.5 实现的 MCP Tools

| Tool | 描述 |
|------|------|
| `start_combat` | 开始战斗，投先攻，排序参战者 |
| `get_combat_state` | 获取战斗状态 |
| `attack` | 执行攻击（支持优势/劣势） |
| `cast_spell` | 施放法术（伤害/治疗） |
| `end_turn` | 结束回合，推进到下一个参战者 |
| `end_combat` | 结束战斗，生成统计报告 |
