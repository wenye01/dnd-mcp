## 技术约束

1. **LLM**: 使用 GLM-4.7-Flash
   - API 端点: `https://open.bigmodel.cn/api/paas/v4/chat/completions`
   - 模型: `glm-4.7-flash`

2. **协议**: MCP (Model Context Protocol)
   - 传输: stdio（推荐）或 HTTP

3. **存储**: Redis + PostgreSQL
   - 与 Server 共用存储

4. **部署**: 单机部署优先

## 验收标准

### 功能验收

- [ ] 用户可通过自然语言创建战役
- [ ] 用户可通过自然语言创建角色
- [ ] 用户可通过自然语言进行战斗
- [ ] 用户可通过自然语言投掷骰子

### 非功能验收

- [ ] LLM 响应时间 < 10s
- [ ] 工具调用延迟 < 500ms
- [ ] 错误情况有友好提示
