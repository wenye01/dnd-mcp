---

## 里程碑依赖说明

| 里程碑 | 依赖 | 说明 |
|--------|------|------|
| M1 | 无 | 基础设施，其他所有里程碑依赖 |
| M2 | M1 | 需要 MCP 框架和数据库 |
| M3 | M2 | 需要战役和游戏状态 |
| M4 | M3 | 需要角色属性 |
| M5 | M4 | 需要骰子系统 |
| M6 | M4 | 需要骰子（移动检定） |
| **M6.5** | **M6** | 需要地图模型和存储 |
| M7 | M2 | 需要战役上下文 |
| M8 | M4 (可选) | 可独立开发，但与骰子配合更好 |
| **M9** | **M3, M6.5** | 需要角色模型和 Import 框架 |

## 关键路径

```
M1 → M2 → M3 → M4 → M5 (战斗系统完成)
                 ↘
                   M6 → M6.5 (地图系统+导入完成)
                           ↘
                             M9 (角色/物品导入完成)
```

**最短开发路径**: M1 → M2 → M3 → M4 → M5 → M6 → M6.5 → M9

---

## M6.5 与 M9 的职责划分

| 功能 | 里程碑 | 说明 |
|------|--------|------|
| UVTT 解析 | M6.5 | 地图格式 |
| FVTT Scene 解析 | M6.5 | 地图格式 |
| LevelDB 解析 | M6.5 | Compendium 格式 |
| Map Converter | M6.5 | 地图转换 |
| import_map | M6.5 | 地图导入 Tool |
| import_map_from_module | M6.5 | 模块导入 Tool |
| FVTT Actor 解析 | M9 | 角色格式 |
| Character Converter | M9 | 角色转换 |
| import_character | M9 | 角色导入 Tool |
| Item Converter | M9 | 物品转换 |
| import_items | M9 | 物品导入 Tool |

## 并行开发建议

完成 M4 后可并行开发：
- **路径 A**: M5 (战斗系统)
- **路径 B**: M6 → M6.5 (地图系统+导入)

完成 M6.5 后可并行开发：
- **路径 C**: M9 (角色/物品导入)
- **路径 D**: M7 (上下文管理，独立)

完成 M2 后可并行开发：
- **路径 E**: M3 → M4 (核心链)
- **路径 F**: M7 (上下文管理，独立)
- **路径 G**: M8 (规则查询，可选依赖 M4)
