## 6.5.2 范围

- **包含**:
  - UVTT 格式导入
  - FVTT Scene JSON 导入
  - FVTT Compendium (LevelDB) 解析和导入
  - 地图图片处理（大文件、格式转换）
  - import_map 和 import_map_from_module Tools
- **不包含**:
  - FVTT Actor/Item 导入（保留在 M9）
  - 角色导入功能（保留在 M9）
  - 光源系统导入
  - ActiveEffect 导入
