## 文档维护

### 需要更新的文档
- **使用指南.md**: 当API或功能发生变化时更新
- **整体架构设计.md**: 当架构发生重大变更时更新
- **client/详细设计.md**: 当 Client 实现发生变化时更新
- **server/*.md**: 当 Server 设计发生变化时更新
- **代码规范.md**: 当编码规范调整时更新

### 文档风格
- 使用Markdown格式
- 中文文档
- 代码示例使用英文
- 清晰的目录结构
- 适当的图表和示例
