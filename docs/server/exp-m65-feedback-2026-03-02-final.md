## 修复建议

### 立即修复 (P0)

**1. UVTT name 字段问题** (`map_converter.go:67`)
```go
// 添加默认名称生成
name := opts.Name
if name == "" {
    name = "Imported Map " + uuid.NewString()[:8]
}
```

**2. FVTT Scene 类型修复** (`fvtt_scene.go`)
```go
// 修复类型定义
type FVTTDrawing struct {
    Type      interface{} `json:"type"`      // 可以是 string 或 int
    Height    float64     `json:"height"`    // 改为 float64
    Points    interface{} `json:"points"`    // 可以是 array
    // ...
}

type FVTTScene struct {
    GlobalLightThreshold *float64 `json:"globalLightThreshold"` // 改为 float64
    // ...
}

type FVTTLight struct {
    Bright float64 `json:"bright"` // 改为 float64
    // ...
}
```
