## 查询策略建议

### 1. 明确查询类型
- **规则问题** → PHB（玩家规则）或 DMG（DM规则）
- **怪物数据** → MM
- **角色选项** → PHB 第2-4章 或 元素邪妄
- **法术详情** → PHB 第11章 或 元素邪妄 第2章
- **魔法物品** → DMG 第7章

### 2. 使用关键词搜索
在 PDF 中搜索中文关键词，如：
- "优势" "劣势"
- "豁免"
- "熟练"
- "法术位"
- "专长"

### 3. 交叉参考
某些内容在多本书中都有涉及：
- **战斗规则**: PHB 第9章（详细）vs 新手套组规则（简化）
- **施法**: PHB 第10-11章（完整）vs 新手套组规则（基础）
- **怪物**: MM（完整）vs 模组附录（特定）
