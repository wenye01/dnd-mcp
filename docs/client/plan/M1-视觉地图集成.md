# Milestone 1: 视觉地图集成

> **状态**: ⏳ 待开发
> **依赖**: Server M6（地图系统）
> **任务数**: 7
> **📅 创建日期**: 2026-03-01

---

## 1.1 目标

为 Client 添加视觉地图集成能力，使 LLM (DM) 能够：
1. 调用视觉 MCP 分析地图图片
2. 处理分析结果
3. 将确认的地点数据提交到 Server

---

## 1.2 范围

- **包含**:
  - 视觉 MCP 配置
  - 视觉 MCP 客户端（调用外部视觉 MCP）
  - 地图上传 API
  - 分析结果处理
  - 与 Server 的集成（调用 create_visual_location 等 Tools）

- **不包含**:
  - 视觉 MCP 服务本身（使用外部服务）
  - 地图编辑功能
  - 前端 UI（由前端应用处理）

---

**测试层**:
- 单元测试: 客户端测试（Mock）

#### 实现细节

```go
// VisualMCPClient 视觉 MCP 客户端接口
type VisualMCPClient interface {
    // AnalyzeMapImage 分析地图图片
    AnalyzeMapImage(ctx context.Context, imageURL string) (*MapAnalysisResult, error)

    // DescribeScene 描述指定位置的场景
    DescribeScene(ctx context.Context, imageURL string, x, y float64) (*SceneDescription, error)

    // Close 关闭客户端
    Close() error
}

// MapAnalysisResult 地图分析结果
type MapAnalysisResult struct {
    Summary   string           `json:"summary"`
    Locations []VisualLocation `json:"locations"`
    Terrains  []TerrainArea    `json:"terrains"`
    Landmarks []Landmark       `json:"landmarks"`
}

// VisualLocation 视觉识别的地点
type VisualLocation struct {
    ID          string  `json:"id"`
    Name        string  `json:"name"`
    Type        string  `json:"type"`
    Description string  `json:"description"`
    PositionX   float64 `json:"position_x"`
    PositionY   float64 `json:"position_y"`
}
```

#### 约束条件

- 前置任务: T1-1
- 技术约束: 支持超时、重试
- 边界条件: 服务不可用、分析失败
