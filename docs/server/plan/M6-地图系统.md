# Milestone 6: 地图系统

> **状态**: ⏳ 待开发
> **依赖**: [M4](M4-骰子系统.md)
> **任务数**: 14（原 7 + 视觉扩展 7）
> **⚠️ 涉及 D&D 5e 规则**: 参见 [00-规则要求.md](00-规则要求.md)
> **📅 更新日期**: 2026-03-01
> **📝 变更说明**: 合并视觉能力扩展（方案 B：Client LLM 调用视觉 MCP）

---

## 6.1 目标

实现大地图和战斗地图的完整功能，包含：
1. **Grid 模式**：基于格子的传统地图系统
2. **Image 模式**：基于图片的视觉地图系统（新增）
3. **FVTT 导入**：扩展字段支持

---

## 6.2 范围

- **包含**:
  - 地图数据（Grid 模式 + Image 模式）
  - 地点标记（Location + VisualLocation）
  - Token 管理和移动
  - 地图切换
  - 扩展字段（图片、墙体）
  - 视觉地点存储（接收 Client 提交的数据）
  - 玩家标记（PlayerMarker）
- **不包含**:
  - 地图编辑
  - 视觉分析（由 Client LLM 调用视觉 MCP 处理）

---

## 6.3 设计调整说明

> 详见 [设计调整-大世界地图视觉能力.md](../设计调整-大世界地图视觉能力.md)

### 6.3.1 双模式设计

| 模式 | 适用场景 | 位置表示 | 移动方式 |
|------|----------|----------|----------|
| **Grid** | 战术地图、FVTT 导入 | `{x, y}` 格子坐标 | 基于格子计算 |
| **Image** | 大世界地图、纯图片 | `{position_x, position_y}` 归一化 (0-1) | 地点跳转 |

### 6.3.2 职责划分（方案 B）

```
┌─────────────────────────────────────────────────────────────┐
│                      Client (MCP Client)                     │
│                                                              │
│  LLM (DM) 职责:                                              │
│  1. 调用视觉 MCP 分析地图图片                                 │
│  2. 处理分析结果                                             │
│  3. DM 确认/调整地点                                         │
│  4. 调用 Server 接口存储数据                                 │
└─────────────────────────────────────────────────────────────┘
                           │
                           ▼
┌─────────────────────────────────────────────────────────────┐
│                      MCP Server                              │
│                                                              │
│  Server 职责:                                                │
│  - 存储和管理地图数据                                        │
│  - 存储和管理地点数据（VisualLocation）                      │
│  - 跟踪玩家位置 (PlayerMarker)                               │
│  - 提供 create_visual_location, update_location 等 Tools    │
│                                                              │
│  Server 不负责:                                              │
│  - 视觉分析（由 Client 处理）                                │
│  - MCP 客户端功能                                            │
└─────────────────────────────────────────────────────────────┘
```

## 6.4 FVTT 影响说明

| 变更项 | 原设计 | 新设计 |
|--------|--------|--------|
| Map.Mode | 无 | `MapMode`（grid / image） |
| Map.Image | 无 | `*MapImage`（URL + 尺寸） |
| Map.Walls | 无 | `[]Wall`（墙体数据） |
| Map.ImportMeta | 无 | `*ImportMeta`（导入元数据） |
| VisualLocation | 无 | `[]VisualLocation`（视觉地点） |
| PlayerMarker | 无 | `*PlayerMarker`（玩家标记位置） |
| Token 扩展 | 基础字段 | +Name, Image, Width, Height, Rotation, Scale, Alpha, ActorLink, Disposition, Hidden, Locked, Bar1, Bar2 |
